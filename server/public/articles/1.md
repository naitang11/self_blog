# 我的博客诞生记

花了一晚上用 Vanilla JS 从零搭了这个博客。没有 Hexo，没有 Hugo，就一个 HTML 文件走天下。这篇文章聊聊为什么我要自己造这个轮子。

---

## 为什么要自己写

说实话，Hexo 和 Hugo 都很好用，主题也多。但每次想要改点东西，总要翻主题文档、改配置文件，有时候还碰到主题不兼容的问题。最烦的是本地要装一堆环境，Hexo 要 Node.js，Hugo 要另外一套，文章一多生成还慢。

我就想：**能不能就一个 HTML 文件，打开展示，部署到任意静态托管**？

抱着这个念头，直接开干。

## 技术选型

- **HTML + CSS + JS**：没有框架，没有构建工具
- **Markdown 渲染**：用 `marked.js`（3KB）直接在前端解析
- **代码高亮**：用 `Prism.js`，主题自己写
- **搜索**：原生 JS 实现，不依赖任何后端
- **深色模式**：CSS 变量 + `localStorage` + `prefers-color-scheme`

就这样，几个文件加起来不到 50KB。

## 最大的挑战

其实不是写代码，是**设计**。我不是设计师，颜色搭配、排版、间距，全靠自己慢慢调。参考了一些个人博客，最后选了一个简洁的卡片风格，字体用了 Noto Serif SC 和 Inter，配上蓝色点缀，看起来还算舒服。

深色模式是最费时间的。要保证每一种颜色在亮色和暗色下都协调，不能只是简单地把背景变黑、文字变白。前后调了十几个版本才满意。

## 现在看来

虽然是个"轮子"，但这个轮子完全受我控制。想加什么功能，直接改代码，不用等主题更新。这大概就是自己造轮子的意义吧。

后续打算加一些功能：标签系统、文章分类、订阅 RSS、生成 sitemap。如果有人感兴趣，也可以开源出来大家一起玩。
